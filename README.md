# :space_invader: Laser Defender :space_invader:

Single level, endless play where player shoots enemies for points until health reaches zero and finally game over
 
## :video_game: Game Design

**Player Experience:**

- Frantic

**Core Mechanic:**

- Shoot Enemies
- Dodge Bullets

**Theme:**

- Space Wars
